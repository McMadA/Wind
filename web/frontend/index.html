How to get a Folder ID

To get a folder ID from Google Drive or OneDrive:

  1. Open your cloud drive in the browser.
  2. Navigate to the specific folder you want to sync.
  3. Look at the URL in your browser's address bar.
  4. The Folder ID is the long string of letters and numbers at the end of the URL.
  5. Example:
    https://drive.google.com/drive/folders/1A2b3C4d5E6f7G8h9I0j
